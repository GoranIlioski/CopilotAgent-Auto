Ова е за агент кој што автоматски се активира кога во SharePoint ќе се внеси документ. 

Ситуацијата:
Имаме екесл документ во кој што ставаме наш трошок за тековниот месец
При тоа сакаме да направиме агент кој што согласно трошковникот, ќе го пополни екселот.
Како што се поставува документот во SharePoint така тој се разгледува и се пополнувач.
